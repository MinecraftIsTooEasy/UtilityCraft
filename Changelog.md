# v1.0.4

---

- 添加黑曜石船，可以在岩浆上划船，需要用铁以上镐子破坏
- 添加133格远古金属储物箱，2000点爆炸抗性

- 添加镐尖斧系列工具，镐子斧头铲子功能三合一，可以附魔:效率/耐久/砍树/时运/穿透/精准采集
- 添加军刀和迅捷剑系列武器，有粒子特效和音效

- 迅捷剑：可以无视部分无敌帧从而使“攻速”变快，幅度11.11%，并且无视部分护甲，如下
- 输入：迅捷剑材质等级 x ，护甲材质等级 y
- 计算差值 delta = x - y
- 护甲穿透：
- delta >= 3.0 → 1.00（100%）
- delta >= 2.0 → 0.75（75%）
- delta >= 1.0 → 0.50（50%）
- delta >= 0.0 → 0.25（25%）
- delta >= -1.0 → 0.10（10%）
- 否则 → 0.00（0%）

- 材质等级：
- 铜/银/金 = 0.0
- 锈铁 = 0.5
- 铁 = 1.0
- 远古金属 = 1.0
- 秘银 = 2.0
- 艾德曼 = 3.0

- #ITF-RB
- 镍 = 1.0
- 钨 = 2.5
- 乌鲁金属 = 6.0

- #ITE
- 附魔金属 = 5.0
- 振金 = 6.0

- 军刀：初始有1点横扫伤害，在附魔上添加的“横扫之刃”附魔后，采用原版的横扫伤害计算公式,如下
- 1 + 武器基础伤害 × (等级 / (等级 + 1))
